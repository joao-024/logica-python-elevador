# logica-python-elevador
Sistema de Elevadores em Python

Este projeto simula o funcionamento de dois elevadores em um prédio utilizando a linguagem Python.

O usuário informa o andar de origem e o andar de destino. O sistema verifica se os andares são válidos, calcula qual elevador está mais próximo da chamada e seleciona automaticamente o melhor para realizar o atendimento. Após buscar o usuário, o elevador se desloca até o andar desejado e sua posição é atualizada para as próximas chamadas.

Funcionalidades
Cadastro de andares válidos do prédio;
Chamada de elevador por andar de origem;
Escolha automática do elevador mais próximo;
Simulação do deslocamento do elevador;
Atualização da posição dos elevadores após cada viagem;
Menu interativo para múltiplas chamadas.
